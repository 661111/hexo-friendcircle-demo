# hexo-friendcircle-demo

这个仓库用于存放hexo友链朋友圈的演示页面以及js和css主题文件

如果你想让你的友链朋友圈css魔改被此仓库收录，请将你的css放入Theme文件夹中并提交PR（注意是**完整不基于其他css主题的文件**，我们会进行独立验证）
